Features

TagSEA is a lightweight tool designed to help you to navigate, and tag files and source code within Eclipse. It offers:

  • Tight integration with the Eclipse Java editor, allowing you to quickly tag your code using Java comments.
  • Hierarchical structuring of tags, to help you keep them organized.
  • A Tag View that displays your tags, and the context of where they occur in your files (their waypoints).
  • A Route View that helps you to organize tags and waypoints in a way that makes sense to you.
  • <Ctrl-space> will display code hints that help you along while you are tagging your source files.
  • Interactive refactoring of tags and waypoints useful for quick restructuring.
