Download and Installation

  1. To install TagSEA first make sure Eclipse is closed. Also remove any previous versions of TagSEA from your eclipse/plugins directory (e.g. ca.uvic.cs.tagsea_0.0.99.jar).
  2. Unzip the TagSEA zip file into your eclipse installation directory and start Eclipse. If you need to download TagSEA, it can be found here.
  3. You are now ready use TagSEA. See the Usage instructions to find out how.
